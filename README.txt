Features like Back InStock Notifications, Save for Later and Sale Countdown Timer for WooCommerce.

== Description ==

Streamline your operations with our multipurpose WordPress plugin. Features like Back InStock Notifications, Save for Later and Sale Countdown Timer.

== Installation ==

1. Download the plugin file to your computer and unzip it
2. Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation's wp-content/plugins/ directory.
3. Activate the plugin from the Plugins menu within the WordPress admin.

== Frequently Asked Questions ==

= How to use Back InStock Notifications =

After installing the plugin, either a "Notify Me" button(for logged in customers) or a form with name and Email(for guests) will be displayed on all out of stock products/variations. When you update the stock, a email notification is sent to all users who requested to be notified when the product/variation is back in stock.

= How to use Save for Later feature =

After installing the plugin, a "Save for later" text is displayed under quantity selectore for each item in the cart. Customers can click on the text to save their cart items for later. They can then move the items back to cart later or delete them altogather.

= How to use Sale Countdown Timer =

After installing the plugin, Simply set end date for sale in WooCommerce product screen.
